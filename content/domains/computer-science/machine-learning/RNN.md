# RNN

A recurrent neural network is a neural network architecture for sequence data. It processes items in order and carries a hidden state forward, so earlier inputs can influence later predictions.

RNNs are useful when order matters, such as text, speech, sensor streams, or [[Time series]] data. Their main limitation is that long dependencies are hard to preserve through many recurrent steps, which is why variants such as [[LSTM]] were introduced.
