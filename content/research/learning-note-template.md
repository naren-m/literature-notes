# Learning Note (template)

The original Notion page was blank — a placeholder for the category "learning note". Preserved here as a template so the shape is reusable.

## Template

```markdown
## What I'm learning

<topic / goal in one line>

## Key ideas so far

- <bullet 1>
- <bullet 2>

## Open questions

- <thing I don't yet understand>

## Next step

<the smallest concrete action that moves me forward>
```

## Usage

Use this when a topic is new enough that a full research note would be premature. Commit the note, iterate on it as understanding grows, and promote to a proper research note (or permanent note in `content/domains/`) when the ideas stabilise.
