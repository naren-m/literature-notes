# Algodeck — Awesome Flashcards for Algorithms

Literature stub capturing a pointer to the Algodeck repository — a curated set of algorithm and data-structure flashcards maintained by Teiva Harsanyi.

## About

Algodeck is a public GitHub collection of flashcards covering data structures and algorithms: arrays, linked lists, trees, graphs, dynamic programming, sorting, complexity, and more. It's designed as a spaced-repetition resource for interview prep and general algorithm review.

Repository: <https://github.com/teivah/algodeck>
