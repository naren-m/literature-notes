# Memory Palace (Method of Loci)

One-line idea from a near-empty Notion stub: **construct an imagined palace and populate it with weird, striking images** so the things you want to remember stick.

## The core claim

The brain is wired to remember **images** and **places** far better than lists of abstract facts. The method of loci (a.k.a. memory palace) exploits this by:

1. Picking a location you know well (childhood home, daily commute route).
2. Walking through it in a fixed order, placing a vivid mental image at each location.
3. Making the images *weird*, *surprising*, *exaggerated*, or *emotionally charged* — bland imagery fades; absurd imagery sticks.
4. To recall, mentally walk the route again and "pick up" each image at its location.

## Why it works

- **Spatial memory** is one of the oldest, most robust memory systems in humans. We evolved to navigate.
- **Visual encoding** provides more retrieval cues than verbal encoding.
- **Distinctiveness** (weirdness) defeats interference — each loaded location is unambiguously different from its neighbors.

## When it's useful

- Sequences of items (speeches, shopping lists, ordered arguments).
- Lists where the *order* matters.
- Domain knowledge that can be chunked into discrete images (e.g., elements of the periodic table, historical dates).

## When it's not

- Genuinely conceptual understanding — palaces store facts, not insights.
- Anything where the underlying pattern is more compact than the list of instances.

## Candidate permanent notes

- **"The brain remembers images and places, not lists."** The one-line rationale for memory-palace techniques.
- **"Weird images stick; bland ones fade."** The distinctiveness heuristic that makes loci work.

## Open threads

- Try building a small memory palace for a specific concrete list (e.g., major phylum in biology) and measure retention at 1 day, 1 week, 1 month.
- Compare memory palaces to spaced repetition (Anki) — they're complementary, not substitutes.
