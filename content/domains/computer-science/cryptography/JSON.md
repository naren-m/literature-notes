# JSON

JSON, or JavaScript Object Notation, is a text format for representing structured data as objects, arrays, strings, numbers, booleans, and null values.

Its practical value is interoperability. A JSON document is easy for humans to inspect and easy for programs in different languages to parse, which is why it is common in web APIs, configuration files, and token formats.

In security-sensitive systems, JSON is only a representation. The trust comes from surrounding controls such as [[Token Authentication]], signatures, schemas, and careful parsing.
