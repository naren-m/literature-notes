## Metadata Guide

### Required Fields
- **title**: Note title (quoted for spaces)
- **date**: YYYY-MM-DD format

### Optional Fields
- **category**: Domain/Subdomain (e.g., "CSE/Cryptography", "sanskrit-lit/Philosophy")
- **tags**: Array of relevant tags [tag1, tag2, tag3]
- **status**: draft | incomplete | complete | archived
- **source**: Reference if from external material (book, article, course)
- **author**: Original author if not you
- **related**: Array of wikilinks ["[[Note1]]", "[[Note2]]"]

### Tag Guidelines
Use lowercase, hyphenated tags:
- `#cryptography` not #Cryptography
- `#design-patterns` not #DesignPatterns
- `#yoga-sutras` not #YogaSutras

Common tags by domain:
- **CSE**: #security, #algorithms, #design-patterns, #cryptography
- **Sanskrit-lit**: #vedic, #yoga, #philosophy, #upanishads
- **Books**: #self-improvement, #biography, #technical, #fiction
