# Bantu reethi Kolu

__Pallavi__: baNTu rIti koluv(i)yya(v)ayya rAma

- __Anupallavi__: 
	- tuNTa viNTi vAni modalaina [[mada]](A)dula goTTi nEla gUla jEyu nija (baNTu)
		- tunta vinti vaani-> one with sugar cane bow - [[Manmadha]], personification of [[kaama]]
		- tuNTa viNTi = tunta -> piece, joint, segemnt
		- [[mada]] adula  -> [[mada]] aadula, etc.,
		- gotti kula chese niha bantu reethi kolu ( [[hanuma]] )


kamam tho modalai, [[mada]] adla nit gotti kula chesea, nija bantu (hanuman)

tuNTa viNTi vAni modalaina mad(A)dula --> [[kaama]], krodha, moha, [[mada]], matsralu
