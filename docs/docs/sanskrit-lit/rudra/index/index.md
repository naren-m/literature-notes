# Rudra

रुक्  द्रावयथि  इथि  रुद्राह
रु  द्रावयथि  इथि  रुद्राह

Ruk Dra -> Ruk draavayathi ithi Rudraha

A flow of pain and suffering

Ru Dra -> Ru Draavayathi ithi Rudraha

Ru -> Knowledge
Dra -> Flow
