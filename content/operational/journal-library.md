# Journal library — Notion Journal database

Consolidated index of the Notion **Journal** database (`collection://c7e57818-d637-4428-bf63-4973bed188f0`). The database holds daily entries, special events, learning-journal templates, and random-thought pages, dominated by dated entries from mid-2020 through late 2020 with sparse later additions.

## Why a library file

Daily journal entries are **personal** — they contain lived-experience observations (family moments, learning what-I-did-today notes, a few story-worthy moments), not reference material. Migrating each of 60+ dated entries into its own markdown file would be high-churn and low-utility; most entries are a paragraph or two. The per-entry content is better preserved by keeping the Notion archive intact and indexing it here, so future permanent notes can cite the specific daily entry by Notion ID if a particular day matters.

Same dual-track pattern as [[reading-library]], [[courses-library]], [[meeting-notes-library]], [[projects-library]], [[tutorials-library]].
