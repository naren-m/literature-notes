# Hello World

A "Hello, World!" program is the smallest useful proof that a programming environment can compile or run code and produce visible output.

Its value is diagnostic rather than intellectual. Before solving a real problem, it confirms that the toolchain, editor, runtime, permissions, and output path are wired correctly. That makes it a practical first test for a new language, machine, framework, or deployment target.
