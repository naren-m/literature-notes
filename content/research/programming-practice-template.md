# Programming Practice (template)

Empty scaffold from Notion for writing up a programming-practice problem. Preserved here as a reusable template rather than as content.

## Template

```markdown
## Problem

<problem statement, constraints, input/output examples>

## Solution

<approach, code, edge cases>

## Time Complexity

<big-O analysis, tradeoffs vs. alternatives>
```

## Usage

Copy the block above when starting a new programming-practice writeup; drop it into `content/research/<problem-slug>.md` and fill in each section. Pairs well with [[outlining-the-problem]] as the general problem-solving checklist.
