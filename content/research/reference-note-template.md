# Reference Note (template)

Minimal scaffold from Notion for a "reference" / resource-style note — a stub that just captures an external link plus context. Preserved as a template.

## Template

```markdown
## Resource Note

- **Source:** <URL>
- **Why saved:** <one-line reason you want to find this again>
- **Key takeaway:** <single-sentence distillation once read>
```

## Usage

Use when bookmarking an article, paper, or video without (yet) writing a full literature note. Once the resource is read and digested, promote the file to `content/media/articles/` (or `videos/`, `books/`, etc.) with a proper frontmatter `source:` and follow the literature-note shape used elsewhere in the vault.
